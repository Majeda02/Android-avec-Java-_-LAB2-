### "# Android-avec-Java-_-LAB2-" 

#### Objectif :
  Créer une application qui calcule le montant total des impôts locaux selon :
   * la surface de la maison (en m²) ;
   * le nombre de pièces ;
   * la présence ou non d’une piscine.
